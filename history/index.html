（前）聚會歷史

最初的原點

繼 PyCon TW 2012 、COSCUP 2012 後，我們發現台北的 Python 使用者與議題已經多到不聚不可、不分享不行了，於是我們開始來試試，每個月固定來辦個聚會，歡迎來分享、研究、把玩、認識 Python 五花八門的應用與技術，之道。

2013-02 以及之後的活動都在 Taipei.py 的 Past Meetups 上囉。
