簡介

Taipei.py 又稱【台北 Python 使用者群組】，每個月會舉辦 Seminar（討論會）讓大台北的 Python 愛好者或對 Python 有興趣的人，能藉由難得的機會一起分享、研究、把玩、認識它五花八門的應用與技術，以及 Python 之美。

月會之外，由 Andy Dai 主辦的隔週 Python Web Meetup，也有與 Django Girls Taipei 合辦的聚會。PyCon Taiwan 的成員也會在聚會中出沒。

到 Taipei.py Meetup 上 RSVP: YES 參加下一個有趣的活動吧！
